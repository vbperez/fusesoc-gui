For create FORMS with python we need to install WXPYTHON. 

In linux we will need the package python-wxgtk2.8  

For Debian distribution it is enough to sudo apt-get install python-wxgtk2.8

In Gentoo:  emerge wxpython

For RedHat, Fedora, Mandriva... yum install wxPython 

You can download the repositories for your distribution here: or dowload here: http://www.rpmfind.net/linux/rpm2html/search.php?query=wxPython

For  Windows or Mac OS

Download the installer en www.wxpython.org/download.php
